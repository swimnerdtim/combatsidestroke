# NSW.navy.mil Design Analysis

Based on the Naval Special Warfare (NSW) official site, here are key design elements to incorporate:

## Color Palette
- **Navy Blue** (primary): Deep, authoritative navy
- **Gold/Yellow** (accent): Naval insignia gold
- **Black/Charcoal**: Text and depth
- **White**: Clean backgrounds, text on dark
- **Red**: Minimal accent for CTAs

## Typography
- **Headers**: Bold, uppercase, military-style
- **Body**: Clean, highly readable sans-serif
- **Font weight**: Heavy emphasis on bold weights
- **Letter spacing**: Wide tracking on headers

## Design Elements
1. **Full-bleed hero images** - Dramatic, high-quality photos
2. **Strong contrast** - Dark/light sections alternating
3. **Minimal ornament** - Functional, not decorative
4. **Grid-based layout** - Structured, organized
5. **Clear hierarchy** - Obvious visual priority
6. **Official insignias** - Trident, NSW logos (appropriate use only)

## UI Patterns
- **Navigation**: Simple, horizontal, sticky
- **Buttons**: Rectangular, solid colors, uppercase text
- **Cards**: Clean edges, subtle shadows
- **Icons**: Minimal, functional
- **Spacing**: Generous whitespace, breathing room

## Photography Style
- High-contrast, dramatic lighting
- Action shots in water/training
- Close-ups of gear/equipment
- Team/group shots showing camaraderie
- Underwater shots (powerful)

## Tone
- **Professional**: Not flashy or commercial
- **Authentic**: Real photos, real stories
- **Serious**: Mission-focused
- **Inspiring**: Without being cheesy

## What to Avoid
- Bright, saturated colors
- Playful fonts or illustrations
- Rounded, soft design elements
- Excessive animations
- Commercial/sales-y language

## Priority Updates for Combat Side Stroke Site
1. Darken the color scheme (more navy/black)
2. Use bolder, heavier fonts
3. Add gold accents sparingly
4. Increase contrast everywhere
5. Make CTAs more rectangular/bold
6. Add authentic training photos
7. Simplify navigation
8. Increase font sizes for authority
